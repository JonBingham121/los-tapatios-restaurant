# Los Tapatios Mexican Restaurant

Multi-location restaurant website for Los Tapatios Mexican Restaurant, serving authentic Mexican cuisine in St. George UT, Kanab UT, and Flagstaff AZ.

## Tech Stack

- **Next.js 14+** - React framework with App Router
- **TypeScript** - Type-safe code
- **Tailwind CSS** - Utility-first styling
- **SEO Optimized** - Schema.org markup, sitemap, robots.txt

## Features

- 🌮 Multi-location restaurant site
- 📱 Fully responsive mobile-first design
- 🎨 Warm Mexican color palette (red, orange, yellow, blue, gold, green)
- 🔍 SEO/AEO/GEO optimized with Restaurant and LocalBusiness schema
- 📍 Location-specific pages with structured data
- 📋 Full menu with pricing
- 🎉 Catering information and packages
- 👑 VIP Club signup
- 📞 Contact forms and location details

## Pages

- **Home** - Multi-location hero, signature dishes, reviews
- **Locations** - Overview of all 3 locations
  - St. George, UT
  - Kanab, UT
  - Flagstaff, AZ
- **Menu** - Full menu with appetizers, fajitas, tacos, specialties, drinks
- **Catering** - Packages and add-ons for events
- **VIP Club** - Rewards program signup
- **Contact** - Contact form and FAQ
- **Order Online** - Order links for all locations

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deployment

This site is designed to deploy to Vercel with zero configuration.
