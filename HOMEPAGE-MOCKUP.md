---

## Color Key

- **Red (#D32F2F)** - Primary buttons, headings, branding
- **Orange (#FF6F00)** - Hover states, gradients
- **Yellow (#FBC02D)** - Gradient accents, highlights
- **Blue (#1976D2)** - Secondary buttons, "Why Locals Love" section
- **Cream (#FFF8DC)** - Alternate section backgrounds
- **Charcoal (#333333)** - Footer, text
- **White** - Main backgrounds, button text

---

## Mobile View (Stacked)

On mobile, everything stacks vertically:
- Hamburger menu replaces nav links
- Location cards stack one above the other
- Signature favorites stack 2x2 grid
- All text centers and scales appropriately

---

## Interactive Elements

**Hover Effects:**
- Buttons change from red → orange on hover
- Location cards lift with shadow
- Nav links underline on hover

**Sticky Navigation:**
- Navbar stays at top when scrolling

**Responsive:**
- Desktop: Multi-column layouts
- Tablet: 2-column layouts
- Mobile: Single-column stacked

---

This is what you'll see when you run `npm run dev` or deploy to Vercel! 🌮
